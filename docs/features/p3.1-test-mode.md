# P3.1 Test Mode

Hub-level toggle for quick testing of all questionnaires.

## Behavior

When enabled:
1. Questionnaires auto-fill random answers on load
2. Skip directly to results/analysis screen
3. No manual input required

## UI

Hub header toggle:
- Toggle switch or button in hub header
- Visual indicator when active
- Persists via URL param `?testmode=1`

## Implementation

Create `lib/testmode.js`:
```javascript
// Check if test mode is active
isTestMode() → boolean

// Get URL with test mode param
getTestModeUrl(baseUrl) → string
```

## Integration

Hub:
- Add toggle to header
- Pass `?testmode=1` to questionnaire links when active

Each questionnaire:
1. Import `lib/testmode.js`
2. Check `isTestMode()` on init
3. If true: fill random answers, show results immediately
