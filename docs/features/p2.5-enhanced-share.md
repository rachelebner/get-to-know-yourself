# P2.5 Enhanced Share

Copy/share results in multiple formats.

## Formats

| Format | Clipboard Type | Use Case |
|--------|---------------|----------|
| Markdown | `text/plain` | Notes, code editors |
| Rich Text | `text/html` | Word, email, Google Docs |
| Image | `image/png` | Chat, presentations (optional) |

## UI

Replace current "העתק תוצאות" with dropdown or menu:
- "העתק כטקסט" (Markdown)
- "העתק מעוצב" (Rich Text)
- "העתק כתמונה" (Image - if implemented)
- "שתף" (Web Share API - mobile only)
where this sharing should support all expected formats by the mobile device (standard implementation)
## Implementation

Create `lib/share.js`:
```javascript
// Copy as plain text (current behavior)
copyAsMarkdown(text)

// Copy as HTML (pastes formatted)
copyAsRichText(html)

// Copy as image (canvas screenshot)
copyAsImage(element) // optional

// Native share (mobile)
shareNative(title, text)
```

## Integration

Each questionnaire:
1. Import `lib/share.js`
2. Replace copy button with share menu
3. Build HTML version of results for rich text copy
